пример для начального заполнения файла миграций
1. Создать в постгресе 2 БД основная и source пустые
2. Запустить в режиме diff, оно создаст в source структуру, и ругнется. Запуск в данном случае нам не нужен, только создание таблиц.
3. Запустить с командной строки mvn liquibase:diff. Создастся файл src/main/resources/db/db_last_diff.xml. Это и есть миграция, ее можно скопировать в файл миграций, например, changelog-1.1.xml.
4. Проверить что все работает, запустив в prod режиме.

По аналогии можно сделать миграции путем сравнения с предыдущей структурой в БД source.
P.S. для postgresql требуется явное создание генераторов первичных ключей с помощью соотв. аннотаций в java коде проекта.
